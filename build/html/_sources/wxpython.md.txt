# WxPython

## Install from a wheel

```bash
pip install /home/jeff/projects/wx-tmp/wxPython-4.2.1-cp312-cp312-linux_x86_64.whl
```

NB may have to change the pyhon version in the wheel file name (cp12). To check supported tags rin

```bash
pip debug --verbose
```

## Installation

Using [these instructions](https://wxpython.org/blog/2017-08-17-builds-for-linux-with-pip/index.html) ...


### Necessary libraries

| Library|Manjaro name|
| --- | --- |
| python-dev | [Not required in Arch](https://unix.stackexchange.com/questions/499786/ways-to-install-python3-dev-or-python3-devel-on-arch-linux) |
| gtk | gtk3 & gtk4 |
| gstreamer | gstreamer |
| gstreamer-plugins-base | gst-plugins-base |
| glut | freeglut |
| libwebkitgtk | webkitgtk-6.0 |
| libjpeg | libjpeg-turbo |
| libpng | libpng |
| libtiff | libtiff |
| libsdl | sdl2 & sdl12-compat |
| libnotify | libnotify |
| libsm | libsm |


### The --enable-shared flag

Build the virtual environment with

```bash
 env PYTHON_CONFIGURE_OPTS="--enable-shared" pyenv install 3.x.x
```
