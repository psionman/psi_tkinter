    WxPython¶

    Install from a wheel¶

    pip install /home/jeff/projects/wx-tmp/wxPython-4.2.1-cp312-cp312-linux_x86_64.whl

    NB may have to change the pyhon version in the wheel file name (cp12). To check supported tags rin

    pip debug --verbose

    Necessary libraries¶

    Library Manjaro name

    The –enable-shared flag¶

    Build the virtual environment with

     env PYTHON_CONFIGURE_OPTS="--enable-shared" pyenv install 3.x.x

    Build on Snowdon¶

    Built on laptop and produced a wheel file at /home/jeff/projects/wx-tmp/wxPython-4.2.1-cp311-cp311-linux_x86_64.whl

    Installed wheel on desktop using

    pip install /home/jeff/projects/wx-tmp/wxPython-4.2.1-cp311-cp311-linux_x86_64.whl
